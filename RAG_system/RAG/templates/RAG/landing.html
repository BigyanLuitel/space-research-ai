How it works

Three steps.
Zero friction.

From raw data to precise answers in under a minute.

01 —

Upload your file

Drop in a PDF mission report or a CSV satellite database. AstroQuery validates and indexes it automatically.

02 —

Data gets indexed

Your file is chunked, embedded using sentence transformers, and stored in a local ChromaDB vector store.

03 —

Ask anything

Type a question in plain English. The system retrieves relevant chunks and generates a precise, cited answer.

Retrieval

Grounded answers only

Every answer is retrieved from your uploaded document. The system will not hallucinate data outside the file — if it's not there, it says so.

Formats

PDF and CSV support

Ingest mission reports, research papers, and technical documents as PDFs — or satellite orbital databases as CSV files.

Memory

Conversation history

Ask follow-up questions naturally. The chat remembers previous turns so you can drill down without repeating context.
